# Spring Boot CLI
## SDKMAN
- 安装sdkman

```bash
$ curl -s "https://get.sdkman.io" | bash
$ source "$HOME/.sdkman/bin/sdkman-init.sh"
```

- 关闭并重新打开terminal

```bash
$ sdk install springboot
$ spring --version
```

- 初始化一个项目
```bash
$ mkdir demo
$ cd demo
$ spring init
```
